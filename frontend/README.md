# ClinIQ - React Frontend

Modern React-based UI for the Clinical RAG application.

## Setup

1. Install dependencies:
```bash
cd frontend
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Features

- **Document Upload**: Upload PDF, DOCX, or TXT files
- **Chat Interface**: Ask questions about your documents
- **Configuration Panel**: Set API key and search options
- **Modern UI**: Clean, responsive design with Tailwind CSS
- **Real-time Feedback**: Toast notifications for user actions

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router
- React Hot Toast
- Lucide React Icons
